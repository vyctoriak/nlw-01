# ECOLETA

Projeto feito durante a semana do meio ambiente. Seu objetivo é conectar as pessoas aos pontos de coletas de resíduos. 

## Getting Started

### Installing

Para instalar os pacotes e rodar na sua máquina, siga as intruções abaixo:

```
npm install
```

E em seguida:

```
npm start
```
